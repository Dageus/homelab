# XPEnology

[XPEnology](https://xpenology.org/) is a "cracked" version of [Synology](https://www.synology.com/en-global)'s and their NAS OS [DSM](https://www.synology.com/en-us/dsm), allowing you to run their custom built OS on your own hardware.
