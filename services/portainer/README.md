# Portainer

Portainer is a container management platform that enables you to track and orchestrate your containers in any environment.

### Repo

[https://github.com/portainer/portainer](https://github.com/portainer/portainer)

### Website

[https://www.portainer.io/](https://www.portainer.io/)

## Guide

Portainer is very powerful, and you can manage ALL your containers there.

The thing is, for Portainer to have edit access to your containers, they need to be deployed through Portainer, which isn't always useful, so I don't use this feature a lot.

I mostly use Portainer to monitor my environments.

### Adding an environment

After adding [Portainer Agent](../portainer_agent/README.md) to your environment, you can add it by using the IP address of the environment it's deployed in and the portainer agent's port: `9001`.
