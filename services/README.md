Here you'll find every service I've ever deployed (even ones that are unused just for history purposes), with a small `README.md` explaining it's purpose and some useful links for configuration/customization and sources for where I've gotten the information.

Here is a small summary of every service I have:

- `Adguard Home` - DNS server that serves as an adblocker

- `Cloudflared` - Cloudflare daemon that allows the usage of [cloudlfare tunnels](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/)

- `glance` - small web dashboard for all your information, from RSS feeds to news and entertainment

- `Immich` - image storing app with a wide variety of features.

- `Nextcloud` - TODO

- `Nginx-Proxy-Manager` - Reverse proxy with Let's Encrypt TLS certificates available

- `Portainer` - Dashboard and Manager for all your docker containers

- `Tailscale` - VPN service with central hub that allows easy connection and management (centralized, for uncentralized and fully self-hosted, look at [Wireguard](https://www.wireguard.com/))

- `Traefik` - Reverse Proxy and Container orchestrator with load balancing

- `Watchtower` - Automatic updater for all your docker containers

- `wg-easy` - Dashboard for managing Wireguard tunnels

- `wireguard` - VPN Tunnel with state-of-the-art security

- `Vaultwarden` - Open-source version of Bitwarden. Password manager

- `paperless-ngx` - Document scanner with indexer and tags

- `XPEnology` - Bootloader for Synology's DSM NAS software

- `*arr stack` - To be Implemented
