# Wireguard

Wireguard is the most advanced and secure VPN solution on the market, to the point that every VPN provider you see on the internet is probably a wireguard wrapper.

Even [Tailscale](../tailscale/README.md) is a solution built on top of Wireguard.

Safe to say that if you want hands-on control on your VPN and how you set it up: Wireguard is the solution.
