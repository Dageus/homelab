# Actual Budget

Actual Budget is a self-hosting finance managing app, where you can safely and privately manage your finances using the [Envelope Method](https://en.wikipedia.org/wiki/Envelope_system).

### Repo

[https://github.com/actualbudget/actual](https://github.com/actualbudget/actual)

### Website

[https://actualbudget.org/](https://actualbudget.org/)

## Guide

Actual Budget is a very convenient tool, but it has its limitations. To connect you bank account (if you're in Europe), Actual suggested using [GoCardless](https://gocardless.com/), but since July 2025, they don't accept new accounts, so we'd need a new alternative. Some people have turned to [SimpleFIN](https://beta-bridge.simplefin.org/), but me, I decided to do it manually every month.

This might be a big turn off for some people and it's unfortunate that this feature doesn't exist anymore, but it doesn't take away from its usability.
