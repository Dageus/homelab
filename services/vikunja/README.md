# Vikunja

Vikunja is a project-manager software that has Kan-Ban boards and other useful features such as sprints and calendars.

I used to use it to organize my work.

## Repo

[https://github.com/go-vikunja/vikunja](https://github.com/go-vikunja/vikunja)

## Website

[https://vikunja.io/](https://vikunja.io/)

## Guide

Vikunja, overall, wasn't a good experience. From buggy "Unauthorized" errors on the latest version, to "Internal Server Error" errors when downgrading to the more stable 0.26 version, it took more time to set up than to properly use. So unfortunately, I discarded this service, but may welcome it back in the future when it reaches a stable version.

### Database Configuration

Vikunja supports multiple databases, but the default is PostgreSQL. You can change it in the configurations shown in the [website]().


### Reverse Proxy Configuration

You can map Vikunja to use whatever Reverse Proxy you're running on your setup, as also explain on their [website]().
